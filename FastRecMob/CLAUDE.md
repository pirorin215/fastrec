# FastRecMob プロジェクト - Claude Codeへの指示

## 自動ビルドルール

**重要:** このプロジェクトのコードを変更した場合、**必ずビルドを実行してください**。

### 手順

1. コードを変更する
2. **即座にビルドを実行**: `./gradlew assembleDebug`
3. ビルド結果をユーザーに報告

### 例

**ユーザー:** 「関数を修正して」

**Claudeの応答:**
```
✅ 変更完了しました！

ビルドを実行します...
[ビルド結果を表示]
```

## ビルドコマンド

```bash
./gradlew assembleDebug
```
